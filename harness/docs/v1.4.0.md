## P3 — Phase 2 트랙 (T5) — 4 도커 이미지

### typescript (full matcher)

`docker/semantic-typescript/` — Node.js 22 + TypeScript ^5.6:

| 노드 | 추출 |
|------|------|
| `ClassDeclaration` / `InterfaceDeclaration` heritage clauses | `inherits_or_implements` (TypeChecker로 symbol 해소) |
| `NewExpression` (enclosing class) | `creates` |
| `ImportDeclaration` (literal moduleSpecifier) | `imports` |

`node:22-alpine` 베이스 — 약 270MB. node_modules 자동 skip, .d.ts skip.

### kotlin / go / python (stubs)

각 컨테이너: Dockerfile + self-check + 매처 사양 README. NDJSON contract 통일 검증.

|--------|--------|----------|-----------|

| `semantic-go` | golang:1.22-alpine → alpine:3.20 | ~15MB | 2-B (`go/packages` 생성자 함수) |

docker build -t codescan/semantic-kotlin:latest      docker/semantic-kotlin
